# Protocolo de Comunicación - byTokioDemo001

Para optimizar nuestra colaboración, se han definido los siguientes comandos de prefijo para las instrucciones del usuario:

## Comandos
- **`!hablemos*`**: Indica que el usuario desea realizar una consulta, discutir una idea o planificar una sección. El agente debe responder con análisis, sugerencias y diálogo, sin realizar cambios directos en el código a menos que se acuerde.
- **`!hazlo*`**: Indica que el agente tiene luz verde para ejecutar la implementación, escribir código, crear archivos o realizar configuraciones técnicas de forma autónoma.

## Aplicación
Este protocolo asegura que el agente no realice cambios no deseados durante las fases de lluvia de ideas y que sea decisivo durante las fases de ejecución.
