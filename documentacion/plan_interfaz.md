# Plan de Interfaz - Estética Premium

Para cumplir con el requerimiento de un sitio olvidable y de alta calidad, utilizaremos la capacidad `frontend-design`.

## Dirección Estética: "Luxury Tech / Editorial"
- **Tipografía**: Combinación de una fuente Sans-Serif moderna (ej. *Outfit* o *Inter*) con una fuente Serif elegante para títulos (ej. *Playfair Display* o *Cormorant Garamond*).
- **Colores**: Paleta profunda y sofisticada. Fondo Dark Mode refinado o Light Mode con mucho aire (espacio negativo). Acentos en colores vibrantes pero controlados.
- **Micro-interacciones**: Transiciones suaves al navegar entre productos y estados del carrito.
- **Efectos**: Gradientes sutiles (mesh gradients), texturas de ruido fino y sombras suaves para dar profundidad.

## Secciones de la Landing
1. **Hero Section**: Propuesta de valor impactante y botón de "Reservar Turno" (CTA principal).
2. **Calendario de Turnos**: Widget interactivo sincronizado con Google Calendar.
3. **Catálogo de Productos**: Grid dinámico con hover effects premium.
4. **Carrito Lateral (Drawer)**: Acceso rápido sin salir de la página actual.
5. **Formulario de Registro/Login**: Interfaz limpia y minimalista.
6. **Chatbot Flotante**: IA integrada con personalidad de marca.

## Roadmap de Implementación Frontend
1. Configuración de **Tailwind CSS** (o CSS puro si el usuario prefiere) con tokens de diseño personalizados.
2. Setup de **React Router** para las áreas de Admin y Perfil de Usuario.
3. Creación de componentes base: `Button`, `Card`, `Input`, `Modal` siguiendo la nueva estética.
