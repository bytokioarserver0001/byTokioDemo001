# Manual de Capacidades - Proyecto bytokioarserver0001

Este workspace ha sido enriquecido con un conjunto de herramientas y guías especializadas que el agente de IA utiliza para ejecutar tareas complejas con mayor precisión.

## 🎨 Diseño Frontend Premium (`frontend-design`)
Esta capacidad permite al agente ignorar los patrones aburridos y repetitivos de la IA estándar para crear interfaces web:
- **Tipografía Distintiva**: Selección de fuentes con carácter.
- **Color y Contraste**: Paletas de colores vibrantes y modernas.
- **Movimiento**: Animaciones sutiles y micro-interacciones de alta calidad.
- **Estéticas**: Puede trabajar en estilos minimalistas, minimalistas, retro-futuristas, etc.

## 📈 Marketing y Crecimiento
El conjunto de **Marketing Skills** cubre todo el ciclo de vida de un producto digital:

### Estrategia y SEO
- **AI SEO**: Optimización para aparecer en respuestas de ChatGPT, Perplexity y Gemini.
- **Auditoría SEO**: Diagnóstico técnico completo del sitio.
- **Arquitectura del Sitio**: Planificación de jerarquía de páginas y navegación.

### Conversión (CRO)
- **Page CRO**: Mejorar las ventas/registros en landings existentes.
- **Form CRO**: Optimización de formularios para reducir el abandono.
- **Signup Flow**: Mejora del proceso de registro de usuarios.

### Contenido y Copywriting
- **Copywriting**: Redacción persuasiva para homepages y páginas de ventas.
- **Email Sequence**: Creación de secuencias automáticas de bienvenida o retención.
- **Social Content**: Estrategia y creación de posts para LinkedIn, Twitter, etc.

## 🛠️ Herramientas de Desarrollo
- **OpenCode AI**: Integrado globalmente para asistir en la escritura y depuración de código. Se inicia con el comando `opencode`.
- **Supabase**: Backend-as-a-Service conectado para manejar la base de datos, autenticación y almacenamiento de archivos.
